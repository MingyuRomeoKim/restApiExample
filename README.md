# restApiExample

* 개인 로컬에 도커파일을 활용하여 실행합니다.
* mysql 5.7 php 7.4 laravel 8 nginx로 구성되어있습니다.


---

# Postman 테스트 json 링크

* API 테스트는 Postman을 사용합니다.
* https://www.getpostman.com/collections/162420687c6062663afe

# API 명세서

* 이미지는 업데이트가 어려울 수 있습니다.
* API 명세서는 바뀔 때 마다 resources/view/home.blade.php에 명시 해 놓겠습니다.
* 도커파일로 프로젝트 실행 후, 회원가입 후 로그인 시 localhost/home 에서 확인 가능합니다.
